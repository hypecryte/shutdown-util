# Суть программы
---
Чтобы запустить отложенное отключение/перезапуск обычно надо знать время
на которое хочется это сделать, а так же полезть в командрую строку или
команду выполнить, чтобы вбить назойливую команду
Данный скрипт решает данную проблему
# Функции
---
1. Запуск отложенного отключения с учётом конвертации времени
2. Запуск отложенной перезагруки с учётом конвертации времени
3. Отключение отключения/перезапуска
